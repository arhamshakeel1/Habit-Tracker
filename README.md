# Habit Tracker

This is a simple habit tracker made with one HTML file.

It lets you add habits, mark days as done, move between weeks, and see a streak for each habit. The data is saved in the browser, so it stays there after a page refresh.

## How to run

Open `index.html` in a browser.

You can also run it with a small local server:

```bash
npx serve .
```

## Features

The app lets you add a habit, rename a habit by clicking its name, delete a habit, mark days as done, move between weeks, see the current day highlighted, see a streak for each habit, and save data in `localStorage`.

## Tech used

This project uses plain HTML, CSS, and JavaScript.

There is no framework and no build step.

## AI usage

I used AI only in a limited way. It helped me think through the weekly grid layout, the streak logic, and a few small style ideas. I still changed the code myself and made the final choices for the design and behavior.
